# Personal Journey Map Assessment - Data Points Catalog

## Overview

This document catalogs all data points generated by the Personal Journey Map (70-question) assessment, providing a comprehensive reference for:
- Comparison with journal/practice patterns over time
- Cross-validation with self-reported experiences
- Progress tracking across multiple assessment retakes
- Integration with other platform interactions

---

## 1. Thread Capacity Scores

**Source:** Raw assessment responses scored per thread
**Type:** Quantitative
**Range:** 0-40 points per thread (scaled to 0-100%)

| `thread.presence.percentage` | PRESENCE capacity | % | 0-100 |
| `thread.consent.raw` | Raw CONSENT score | points | 0-40 |
| `thread.consent.percentage` | CONSENT capacity | % | 0-100 |
| `thread.memory.raw` | Raw MEMORY score | points | 0-40 |
| `thread.memory.percentage` | MEMORY capacity | % | 0-100 |
| `thread.pause.raw` | Raw PAUSE score | points | 0-40 |
| `thread.pause.percentage` | PAUSE capacity | % | 0-100 |
| `thread.embodiment.raw` | Raw EMBODIMENT score | points | 0-40 |
| `thread.embodiment.percentage` | EMBODIMENT capacity | % | 0-100 |
| `thread.uncertainty.raw` | Raw UNCERTAINTY score | points | 0-40 |

## Comparison Points for Journal/Practice Data

### Thread Focus Alignment
**Compare:** `thread.{name}.percentage` ‚Üê ‚Üí Journal entries mentioning that thread

### Pattern Recognition
**Compare:** `pattern.{id}.behavioralSigns` ‚Üê ‚Üí Journal language patterns showing those behaviors

### Practice Consistency
**Compare:** `hold.{step}.threads` ‚Üê ‚Üí Actual practice log entries for those threads

### Capacity Growth
**Compare:** `thread.{name}.percentage` (Assessment 1) ‚Üê ‚Üí `thread.{name}.percentage` (Assessment 2)

### Pattern Evolution
**Compare:** `patterns.detected.list` (Assessment 1) ‚Üê ‚Üí `patterns.detected.list` (Assessment 2)

### Self-Reported Shifts
**Compare:** `collapse.{thread}.direction` ‚Üê ‚Üí Journal entries showing different collapse patterns

### Development Path Progress
**Compare:** `devPath.items` targets ‚Üê ‚Üí Practice logs showing work on those threads
